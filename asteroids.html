Future Asteroid Classification

By studying and predicting the behavior of asteroids, we can develop strategies for planetary defense and potentially mitigate the impact of hazardous asteroids.

Using NASA datasets that contain information about asteroids, the aim is to discover the combination of variables that directly infleunce the classification of asteroids and train a machine learning model to predict the class based on these variables.

Future Asteroid Classification

  • Role: Data Analytic / Machine Learner
  • Source: Project Github Repository
NASA dataset split into testing/training data. We considered different Machine Learing algorithms, training each model with the training data and using the tesing data to predict the classification, cross-validating the models based on the accuracy score of testing data predictions.
